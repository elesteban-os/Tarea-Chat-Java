- Chat entre varios usuarios.
La aplicación consiste en ejecutar dos o más ventanas diferentes de un chat para enviar entre ellos un mensaje.

Para utilizar la aplicación primero debemos ejecutar el servidor. Para ello, debemos de ir al paquete "server",
buscar "Server.java" y ejecutar ese archivo.
Luego, para abrir ventanas podemos usar el archivo "crearCliente.java" y ejecutar ese archivo las veces que 
queramos. Y ya podremos interactuar entre los diferentes clientes.
